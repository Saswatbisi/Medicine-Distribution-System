The Medicine Distribution System is a full-stack web application designed to facilitate the efficient distribution of medicines from NGOs and individuals to 
people in need. The platform allows users to donate unused medicines and enables verified beneficiaries to purchase medicines at lower prices, helping reduce 
medicine wastage and improve access to essential healthcare resources.
The system focuses on creating a transparent and efficient supply chain between donors, NGOs, and recipients while ensuring proper inventory management and secure 
transactions.

Features
1.Medicine Donation – Users can donate unused or leftover medicines through the platform.
2.Low-Cost Medicine Purchase – Needy individuals can purchase medicines at affordable prices.
3.Real-Time Inventory Management – Track available medicines and update stock dynamically.
4.Secure User Verification – Authentication system to ensure only verified users can access the platform.
5.Distribution Tracking – Monitor the movement and delivery of medicines.
6.NGO Integration – NGOs can manage and distribute donated medicines to communities.
7.User-Friendly Interface – Simple and intuitive interface for donors, NGOs, and buyers.

Problem It Solves
Many people discard unused medicines while others struggle to afford essential drugs. This system helps reduce medicine wastage by redistributing unused medicines
to those who need them at minimal cost.

Technologies Used
1.Frontend: HTML, CSS, JavaScript
2.Backend: Node.js / Express.js
3.Database: MongoDB / MySQL
4.Authentication: Secure user login and verification system
5.Version Control: GitHub
